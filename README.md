# GitHub
Client for the GitHub REST API V3.

Currently supported:
- [x] [Repositories API](https://developer.github.com/v3/repo)
- [x] [Search API](https://developer.github.com/v3/search/)
- [x] [User API](https://developer.github.com/v3/users/)
